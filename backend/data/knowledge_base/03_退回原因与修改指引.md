# 主数据申请退回原因与修改指引

> 版本：V2.0 | 适用范围：荣耀终端有限公司 | 最后更新：2026-02-10

本文档汇总了物料主数据申请最常出现的退回原因，并提供对应的修改指引。申请人收到退回通知后，请对照本文档进行修改。

## 退回原因 1：重复申请

**现象**：系统中已存在相同或高度相似的物料。

**修改指引**：
1. 在 MDM 系统中使用"相似物料查询"功能，确认已有物料
2. 若已有物料完全满足需求，直接复用，无需新建
3. 若已有物料不完全满足但有参考价值，在申请中说明与已有物料的差异
4. 若确认查重有误（非重复），在备注中说明理由并附上差异分析，重新提交

## 退回原因 2：命名规范不符

**现象**：物料名称不符合"功能-规格-特性"三段式格式，或包含缩写、内部代号。

**修改指引**：
1. 确认物料的"功能"类别（如连接器、电阻、IC芯片等）
2. 确认核心规格参数（如型号、尺寸、容量等）
3. 确认区分性特征（如功率、精度、认证等级等）
4. 按"功能-规格-特性"格式重新编写
5. 检查并去除所有英文缩写（如 FPC → 柔性电路板）
6. 参考《物料编码命名规范》（V3.1）确认无误后重新提交

## 退回原因 3：分类类型错误

**现象**：物料分类选择与物料实际类别不匹配。

**修改指引**：
1. 在 MDM 系统中查阅最新版《物料分类体系 V4.0》
2. 根据物料实际属性重新选择三级分类
3. 若无法确定分类，咨询所属领域数据 BP
4. 确认后修改申请中的"物料分类"字段，重新提交

## 退回原因 4：必填字段缺失

**现象**：必填字段未填写或填写信息不完整。

**修改指引**：
物料申请必填字段包括：
- 物料中文名称
- 来源类型（自研/外采/转包/客供/内部调拨）
- 物料分类（三级）
- 基本计量单位
- 申请部门
- 负责工程师

请逐项检查以上字段是否都已填写，补充完整后重新提交。特别提醒：选择"自研"时需同时上传设计图纸；选择"外采"时需填写供应商编码。

## 退回原因 5：单位错误

**现象**：基本计量单位选择不符合物料实际计量方式。

**修改指引**：
1. 电子元器件类：一般为"个"或"片"
2. 线材/线缆类：一般为"米"或"根"
3. 液体/胶水/涂料类：一般为"千克"或"升"
4. 结构件类：一般为"个"或"套"
5. 若不确认，参照同类已有物料的单位设置

## 退回原因 6：业务逻辑矛盾

**现象**：申请信息存在逻辑矛盾，如来源类型与附件不匹配，或分类与名称不匹配。

**修改指引**：
1. 检查"来源类型"与附件是否匹配（自研需附图、外采需填供应商）
2. 检查"物料分类"与"物料名称"中功能段是否一致
3. 检查"基本计量单位"与物料类别是否匹配
4. 修正矛盾项后重新提交，如有特殊业务背景请在备注中说明
