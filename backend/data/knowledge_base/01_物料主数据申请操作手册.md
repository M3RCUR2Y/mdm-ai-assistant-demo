# 物料主数据申请操作手册

> 版本：V2.3 | 适用范围：荣耀终端有限公司 | 最后更新：2026-03-15

## 申请流程概述

物料主数据申请遵循"提交 → 数据BP初审 → 主数据管理员复核 → 系统录入"的四步流程。

1. **提交申请**：申请人通过 MDM 系统填写《物料主数据申请表》，上传相关附件
2. **数据BP初审**：所属领域数据 BP 在 2 个工作日内完成字段规范性审核
3. **主数据管理员复核**：主数据管理团队在 1 个工作日内完成编码规则、去重核查
4. **系统录入**：复核通过后，系统自动生成 PART 码并录入 MDM 主数据库

简单申请（无退回）处理周期为 2-3 个工作日；有退回的申请需 5-7 个工作日或更长。

## 必填字段清单

| 字段名称 | 是否必填 | 说明 |
|---------|---------|------|
| 物料中文名称 | 是 | 格式：功能+规格+特性，如"连接器-USB-C-100W" |
| 物料英文名称 | 否 | 如有英文名称，遵循国际通用命名惯例 |
| 来源类型 | 是 | 枚举值见下方说明 |
| 物料分类 | 是 | 三级分类，参照《物料分类体系 V4.0》 |
| 基本计量单位 | 是 | 如：个、米、千克、升 |
| 申请部门 | 是 | 申请人所属部门 |
| 负责工程师 | 是 | 该物料的负责工程师姓名 |
| 参考图纸编号 | 否 | 如有设计图纸，填写图纸编号 |
| 备注说明 | 否 | 特殊说明或背景信息 |

## 来源类型枚举值

"来源类型"字段用于标识物料的来源属性，必须从以下枚举值中选择：

| 枚举值 | 含义 | 适用场景 |
|-------|------|---------|
| **自研** | 荣耀自主研发设计 | 荣耀研发团队自主设计的定制物料 |
| **外采** | 外部供应商采购 | 从外部供应商直接采购的标准品或定制件 |
| **转包** | 外包/转包生产 | 委托外部厂商代工生产 |
| **客供** | 客户提供 | 客户方提供的物料 |
| **内部调拨** | 内部资产调拨 | 从其他部门或工厂调拨的物料 |

选择"自研"时，需同时上传设计图纸；选择"外采"时，需填写供应商编码。

## 申请退回的 TOP6 原因

根据 2025 年度数据分析，物料主数据申请退回的主要原因如下：

1. **重复申请**（占退回量 22%）：系统中已存在相同或高度相似的物料
2. **命名规范不符**（占退回量 18%）：物料名称不符合"功能+规格+特性"格式，或使用了缩写
3. **分类类型错误**（占退回量 15%）：物料分类选择错误，未参照最新版分类体系
4. **必填字段缺失**（占退回量 14%）：必填字段未填写或信息不完整
5. **单位错误**（占退回量 10%）：基本计量单位选择不符合物料特性
6. **业务逻辑矛盾**（占退回量 8%）：申请信息存在逻辑矛盾，如选择"自研"但未上传图纸

## 申请前相似物料查重

为降低重复申请率，申请人在提交前应通过 MDM 系统的"相似物料查询"功能进行查重：

1. 输入物料关键特征（名称、规格、用途）
2. 系统返回相似度排序的已有物料列表
3. 若存在高度相似的已有物料，可直接复用，无需新建
4. 若确认无相似物料，截图查重结果作为申请附件上传

数据 BP 初审时也会进行二次查重确认。
