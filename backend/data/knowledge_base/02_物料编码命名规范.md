# 物料编码命名规范

> 版本：V3.1 | 适用范围：荣耀终端有限公司 | 最后更新：2026-01-20

## PART 码编码规则

PART 码（物料编码）是 MDM 系统中物料的唯一标识，由系统在申请复核通过后自动生成。PART 码格式为：

```
PART-{年份}-{流水号}
```

示例：`PART-2024-001` 表示 2024 年创建的第 1 个物料。

流水号为 3 位数字，按年度重置。编码由系统自动分配，申请人无需自行编写。

## 物料中文名称规范

物料中文名称必须遵循 **"功能 + 规格 + 特性"** 三段式格式，用连字符 `-` 分隔。

### 三段式定义

| 段落 | 说明 | 示例 |
|-----|------|------|
| 功能 | 物料的核心功能或类型 | 连接器、电阻、电容、IC芯片 |
| 规格 | 关键规格参数 | USB-C、0402、10μF、STM32F103 |
| 特性 | 区分性特征（可选） | 100W、车规级、防水IP67、RoHS |

### 正确示例

- `连接器-USB-C-100W`（功能=连接器，规格=USB-C，特性=100W）
- `电阻-0402-1KΩ±1%`（功能=电阻，规格=0402，特性=1KΩ±1%）
- `电容-0805-10μF-25V`（功能=电容，规格=0805，特性=10μF/25V）
- `IC-电源管理-MP24894`（功能=IC，规格=电源管理，特性=MP24894）

### 禁止事项

1. **禁止使用英文缩写**：不得使用 PCB、FPC、BGA 等缩写代替中文全称（如"印制电路板"、"柔性电路板"、"球栅阵列"）
2. **禁止使用内部代号**：不得使用项目代号、内部 nickname 等非正式名称
3. **禁止省略功能段**：物料名称必须以功能类别开头
4. **禁止使用特殊字符**：除连字符 `-` 外，不得使用 `/`、`\`、`*`、`&` 等特殊字符

## 物料英文名称规范（如适用）

如需填写英文名称，应遵循以下规范：

- 使用国际通用命名惯例
- 与中文名称含义一致
- 禁止使用拼音代替英文

## 编码命名被退回后的修改方法

若申请因"命名规范不符"被退回，请按以下步骤修改：

1. 对照退回原因，确认具体不符合哪条命名规则
2. 重新确认物料的"功能"、"规格"、"特性"三段信息
3. 按"功能-规格-特性"格式重新编写物料名称
4. 检查是否有被禁止的缩写或特殊字符
5. 修改后在申请备注中注明修改内容，重新提交

如不确定物料名称是否正确，可先咨询所属领域的数据 BP 确认。
